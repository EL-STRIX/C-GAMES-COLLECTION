# Screenshot Guidelines

To maintain a high-quality presentation on our GitHub repository, all screenshots should follow these standard guidelines:

1. **Resolution:** Capture at 1280x720 minimum to ensure crisp rendering on high-DPI screens.
2. **Theme:** Use the default dark theme for consistency across all screenshots.
3. **Window State:** Capture the active window cleanly without desktop backgrounds (use Alt+PrtScn or Snipping Tool window mode).
4. **Naming Convention:**
   - `number_guessing_demo.png`
   - `rps_demo.png`
   - `sgw_demo.png`
   - `ttt_demo.png`
   - `launcher_demo.png`

Place all captured files directly in this directory (`assets/screenshots/`). They will automatically be rendered in the main repository `README.md`.
